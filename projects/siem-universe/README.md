# SIEM Universe

Project xây dựng vũ trụ nhân vật thiếu nhi xoay quanh chú mèo cam mập SIEM và nhóm bạn.

## Mục tiêu
- Xây bộ nhân vật nhất quán để dùng lâu dài
- Làm nền cho video ngắn thiếu nhi bằng AI
- Giữ style hình ảnh, tính cách, prompt và câu chuyện không bị trôi

## Nhân vật chính
- SIEM
- BIN
- SIANH
- PIO
- BE
- DOM

## File trong project
- `characters.md`: hồ sơ nhân vật
- `visual-style-guide.md`: quy chuẩn hình ảnh và tính nhất quán
- `prompt-library.md`: prompt gốc để tạo nhân vật ổn định
- `character-sheets.md`: character sheet chi tiết cho từng nhân vật
- `group-scene-prompts.md`: prompt tạo ảnh nhóm và ảnh thương hiệu
- `episode-ideas-01-10.md`: 10 tập ý tưởng đầu tiên
- `story-framework.md`: khung viết tập truyện/video ngắn

## Hướng dùng
1. Chốt nhân vật trong `characters.md`
2. Tạo ảnh mẫu theo `prompt-library.md`
3. Giữ cố định style theo `visual-style-guide.md`
4. Viết câu chuyện theo `story-framework.md`
